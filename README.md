# Masa – Furniture E-Commerce Platform

A full-stack furniture e-commerce web application built with React, Node.js, Express, SQLite, and Stripe.

## Tech Stack

- **Frontend:** React 18, Vite, React Router v6, Axios, Tailwind CSS
- **Backend:** Node.js, Express.js (REST API)
- **Database:** SQLite 3
- **Auth:** JWT + bcrypt
- **Payments:** Stripe

## Getting Started

```bash
# Install server dependencies
cd server && npm install

# Install client dependencies
cd ../client && npm install

# Set up environment variables
cp server/.env.example server/.env
# Add your Stripe keys to server/.env

# Run database setup
cd server && npm run db:setup

# Start the app (from root)
cd server && npm run dev     # API on http://localhost:5000
cd client && npm run dev     # UI  on http://localhost:5173
```
